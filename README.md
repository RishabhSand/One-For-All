## 📖 About

**One For All** is a unified platform designed to simplify student life at **Thapar Institute** by bringing together essential campus services into one place.

Instead of using multiple platforms, students can access everything through a single application.

---

## ✨ Features

### 🤝 SkillSync
Connect with students based on skills they have or want to learn.

- Find mentors
- Learn new skills
- Collaborate on projects
- Build your network

---

### 🏠 Roommate Finder
Find compatible roommates based on lifestyle and preferences.

- Preference matching
- Compatibility score
- Chat system
- Roommate confirmation

---

### 🍽️ Mess Rater *(Coming Soon)*
Review and rate hostel mess food.

- Daily ratings
- Reviews
- Menu feedback
- Popular dishes

---

### 🗺️ Campus Map *(Coming Soon)*
Interactive map of the entire Thapar campus.

- Academic blocks
- Hostels
- Cafeterias
- Sports facilities
- Parking
- Navigation

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/One-For-All.git
```

Move into the project

```bash
cd One-For-All
```

Open any module in your browser.

Example:

```text
roommate-finder/index.html
```
